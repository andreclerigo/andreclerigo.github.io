my personal projects

minecraft website

After my Raspberry Pi 4 purchase I decided to create a Minecraft SMP Server for me and my college friends

But this had to be a special Server, so I decided to create a NodeJS Website that automatically turned ON and OFF the Minecraft Server

Players can also turn ON the Server manually from the website as well as see the Server's Leaderboard for Hours and Deaths


minecraft esp monitor

I also did an ESP monitor for the Website that you can checkout here

Using a Nokia5110 LCD, an ESP-32 and a custom 3D printed case I can have a little monitor beside my computer to always see who is inside the Minecraft Server playing


twitter weather bot

I use Twitter alot and most of my friends do to, so one day I decided to create Weather Daily Bot

The purpose of this bot is to tell the weather forecast to people who want to use it. The bot tags the person who is using it and gives information on the location they chose

To use it, you'll need to tag @WeatherDailyBot on Twitter followed by the Location you want to receive the forecast


Since this project was made initially for me and my friends, all the weather information is in Portuguese
